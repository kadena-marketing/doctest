# Installing Pact on macOS

Setting up Pact on macOS is a simple process that involves installing a few prerequisites and building the Pact project. Follow the steps below to get Pact up and running on your macOS system.

## Prerequisites

Before installing Pact, ensure you have the following tools:

<Tabs>
  <TabItem value="jdk" label="Java Development Kit (JDK)" default>
    Install JDK version 11 or later. You can use Homebrew:

    ```bash
    brew install openjdk@11
    ```

    Verify the installation:

    ```bash
    java -version
    ```
  </TabItem>
  <TabItem value="homebrew" label="Homebrew">
    Install Homebrew for package management (recommended):

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
  </TabItem>
  <TabItem value="git" label="Git">
    Install Git for cloning the repository:

    ```bash
    brew install git
    ```
  </TabItem>
  <TabItem value="maven" label="Maven">
    Install Maven for building the project:

    ```bash
    brew install maven
    ```

    Verify Maven installation:

    ```bash
    mvn -version
    ```
  </TabItem>
</Tabs>

## Installation Steps

<Step stepNumber="1" title="Clone the Repository">
  Open Terminal and run:

  ```bash
  git clone https://github.com/kadena-io/pact.git
  cd pact
  ```

  This clones the Pact repository and changes your current directory to the newly created `pact` folder.
</Step>

<Step stepNumber="2" title="Build Pact">
  Compile the Pact project with Maven:

  ```bash
  mvn clean install
  ```

  This command builds Pact and installs it to your local Maven repository. The process may take a few minutes depending on your system.
</Step>

<Step stepNumber="3" title="Verify Installation">
  To confirm Pact is installed correctly, run:

  ```bash
  mvn exec:java -Dexec.mainClass="pact.main.Main"
  ```

  If Pact is installed properly, this command will execute the Pact main class. You should see output related to Pact's execution.
</Step>

## Usage

Once installed, you can use Pact for developing and testing smart contracts. For more details on usage, refer to the [Pact Documentation](https://github.com/kadena-io/pact).

## Troubleshooting

If you encounter issues during installation or usage:

<Tabs>
  <TabItem value="dependencies" label="Check Dependencies" default>
    Verify that JDK and Maven are correctly installed. You can check their versions:
    
    ```bash
    java -version
    mvn -version
    ```
  </TabItem>
  <TabItem value="github" label="Consult GitHub Issues">
    For common problems and community solutions, visit the [Pact GitHub Issues](https://github.com/kadena-io/pact/issues) page.
  </TabItem>
</Tabs>
